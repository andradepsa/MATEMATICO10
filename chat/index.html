Chat K10 - Aplicativo de Comunicação

Desenvolvido pelo Prof. Paulo Sérgio

Desenvolvido para comunicação privada e segura entre usuários, sem intermediários. O sistema funciona tanto em modo peer-to-peer direto quanto com servidor para grupos.

Comunicação Privada

Converse e envie arquivos com segurança, compartilhando apenas seu ID com contatos.

Salas Compartilhadas

Crie salas para comunicação em grupo usando o modo servidor.

Modos de Operação

Suporta P2P direto, servidor para grupos, e combinações para flexibilidade.
